# Quick Start Guide

## 🚀 How to Run This App

### 1. Prerequisites
- Node.js (version 18.0+)
- npm (comes with Node.js)

### 2. Installation & Setup
```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

### 3. Open in Browser
Visit: **http://localhost:5173/**

### 4. Available Commands
```bash
npm run dev     # Development server (recommended)
npm run build   # Create production build
npm run preview # Preview production build
npm run lint    # Check code quality
```

### 5. Features You Can Explore
- **Dashboard**: Overview of clients and metrics
- **Messages**: Chat with clients
- **Clients**: Manage client profiles
- **Calendar**: Schedule appointments
- **Meal Plans**: Create nutrition templates

### 6. Troubleshooting
- **Port 5173 busy?** Vite will auto-select the next available port
- **Build errors?** Run `npm install` again
- **Missing dependencies?** Delete `node_modules` and run `npm install`

**That's it!** Your nutrition management system is ready to use. 🥗
